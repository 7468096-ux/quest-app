- **Đánh giá Năng lực ĐHQG HCM** — kiến thức liên môn
- **Đánh giá Năng lực ĐHQG HN** — tư duy khoa học
- **Đánh giá Tư duy ĐH Bách Khoa** — toán, lý, hóa
