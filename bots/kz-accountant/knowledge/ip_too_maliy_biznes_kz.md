# ИП и ТОО: регистрация, учёт, отчётность в Казахстане
# Источник: НК РК, ГК РК, Закон о ТОО, kgd.gov.kz, egov.kz
# Актуальность: 2025-2026

---

## 1. ИП — Индивидуальный предприниматель

### Что такое ИП
Физическое лицо, зарегистрированное как предприниматель. Отвечает по долгам всем личным имуществом (в отличие от ТОО).

### Регистрация ИП

**Онлайн (eGov.kz) — бесплатно, 1 рабочий день:**
1. Зайти на egov.kz → «Бизнес» → «Регистрация ИП»
2. Авторизоваться (ЭЦП или через SMS-код)
3. Указать: ФИО, ИИН, ОКЭД (вид деятельности), место нахождения
4. Подписать заявление ЭЦП
5. Свидетельство — электронный документ в личном кабинете

**Через ЦОН — бесплатно, 1 рабочий день:**
- Принести: удостоверение личности + ИИН
- Заполнить заявление
- Получить свидетельство на руки

**Госпошлина:** не взимается

**Ограничения:** ИП не могут заниматься отдельными видами деятельности (банки, страхование, лицензируемые виды — нужна лицензия).

### Как выбрать ОКЭД
ОКЭД — Общий классификатор экономической деятельности (аналог ОКВЭД).
- Поиск ОКЭД: stat.gov.kz → «Классификаторы» → ОКЭД
- Можно указать несколько кодов; первый — основной вид деятельности
- Пример: 69.10.0 — юридическая деятельность, 62.01.0 — IT, 47.11.1 — розничная торговля

### Налоговые режимы для ИП

| Режим | Ставка | Условия |
|-------|--------|---------|
| Упрощённая декларация | 3% от выручки | Доход ≤ 94,5 млн тг/6 мес, ≤ 30 чел. |
| Патент | фиксированная | Только физические лица, ≤ 3 528 МРП/год, ≤ 0 работников |
| Розничный налог | 4% | Перечень видов деятельности |
| ОУР (общеустановленный) | 10% от прибыли | Без ограничений |

### Закрытие ИП
1. egov.kz → «Ликвидация ИП»
2. Закрыть кассовый аппарат (если есть)
3. Сдать финальные декларации
4. Срок ликвидации: 5 рабочих дней

## 2. ТОО — Товарищество с ограниченной ответственностью

### Что такое ТОО
Юридическое лицо. Учредители отвечают по долгам только в пределах своих вкладов в уставный капитал. Аналог ООО в РФ.

### Регистрация ТОО

**Онлайн (egov.kz или ГБД ЮЛ):**
1. Подготовить: устав ТОО, решение об учреждении
2. Размер уставного капитала: минимум **100 МРП** (~393 200 тг в 2026) — для субъектов МСБ минимальный уставный капитал отменён (можно 1 тенге)
3. Подать на регистрацию через egov.kz → «Регистрация юридического лица»
4. Срок: **1 рабочий день**
5. Госпошлина: **9 МРП** (~35 388 тг)

**Что нужно для ТОО:**
- Устав
- Решение единственного учредителя (или протокол учредительного собрания)
- Сведения об учредителях (ИИН/БИН)
- Юридический адрес

### Отличия ТОО от ИП

| Критерий | ИП | ТОО |
|---------|----|----|
| Ответственность | Всем имуществом | Только вкладом в уставный капитал |
| Налогообложение | ИПН 10% (ОУР) или 3% (упрощ.) | КПН 20% (ОУР) |
| Регистрация | Бесплатно | 9 МРП |
| Доверие контрагентов | Ниже | Выше |
| Участники | Только физлицо | Физ- и юрлица, несколько учредителей |
| Дивиденды | Не применяется | ИПН 5% с дивидендов |

## 3. Кассовая дисциплина (онлайн-кассы)

### Когда нужен кассовый аппарат (ККМ)
- При приёме наличных от физических лиц за товары/услуги
- Лимит без ККМ: **100 МРП** (~393 200 тг) в сутки — для ИП на упрощёнке

### Типы кассовых аппаратов
- **Фискальный регистратор** — печатает чеки, подключён к ПК
- **Онлайн-касса (ОФД)** — данные автоматически передаются в КГД

### Реестр ККМ
- Проверить разрешённые модели: kgd.gov.kz → «Реестр ККМ»

### Штрафы за нарушения
- Работа без ККМ: **10% от суммы** неоформленного чека (min 8 МРП)
- Повторное нарушение: 30% (min 30 МРП)

## 4. Bухгалтерский учёт для МСБ

### Кто обязан вести бухучёт
- **ТОО** — обязательно, по МСФО или МСФО для МСБ
- **ИП на ОУР** — ведут книгу учёта доходов и расходов
- **ИП на упрощёнке** — учёт выручки в форме 910

### Упрощённый бухучёт (для ТОО МСБ)
- МСФО для малого и среднего бизнеса (IFRS for SMEs)
- Не требует сложной консолидации
- Возможно ведение в 1С:Предприятие, МойСклад или Buhgalter.kz

### Годовая финансовая отчётность ТОО
- Баланс, ОПУ, ОДС
- Срок представления в КГД/статистику: **31 марта** следующего года

## 5. Документооборот: важные документы

### ЭЦП (электронная цифровая подпись)
- Обязательна для всех ИП и ТОО
- Получить: НУЦ РК → pki.gov.kz или e-Government → «Получение ЭЦП»
- Типы: RSA (для физлиц) и GOST (для юрлиц, рекомендовано)
- Срок действия: **1 год**, затем обновление

### ЭСФ (электронные счета-фактуры)
- Все плательщики НДС обязаны выставлять ЭСФ
- Платформа: esf.gov.kz
- Срок выставления: 20 дней с даты совершения оборота
- Аннулировать или исправить ЭСФ можно в течение 7 дней

### АВР — Акт выполненных работ
- Подтверждает факт оказания услуг
- Можно подписать через ЭЦП (юридически равнозначно бумажному)

## 6. Банковское обслуживание бизнеса в Казахстане

### Топ банки для МСБ
| Банк | Преимущество |
|------|-------------|
| Kaspi Bank | Быстрый онлайн-учёт, интеграция с ККМ, Kaspi Pay |
| Halyk Bank | Крупнейший банк, API для 1С |
| ForteBank | Выгодные тарифы для ИП |
| Jusan Bank | Бесплатное расчётное обслуживание для новых ИП |

### Kaspi Pay для бизнеса
- Приём QR-платежей без терминала
- Деньги зачисляются мгновенно
- Интеграция с чековой лентой (ЭЧ через esf.gov.kz)
- Комиссия: 0,5–1,5% от суммы

## 7. Субсидии и поддержка МСБ

### Программы поддержки
| Программа | Суть |
|-----------|------|
| «Дорожная карта бизнеса» | Субсидирование ставки вознаграждения по кредитам (до 6%) |
| «Экономика простых вещей» | Льготное финансирование обрабатывающей промышленности |
| «Бастау Бизнес» | Обучение и гранты для начинающих предпринимателей |
| «ДКБ-2025» | Гарантирование кредитов МСБ через КГФ (Казахстанский гарантийный фонд) |

### Центры поддержки предпринимательства (ЦПП)
- Бесплатные консультации, обучение
- Найти ближайший: sba.gov.kz
- Есть в каждом городе и районе

### Налоговые льготы для стартапов
- ИТ-компании: КПН 0% (по Astana Hub), ИПН 0% для сотрудников
- Участники СЭЗ: КПН 0%, НДС 0%, таможенные льготы

## 8. Частые ошибки ИП/ТОО

1. **Не сдали форму 200** — даже если нет работников, при упрощённой декларации не нужно, но при ОУР — нужно
2. **Забыли про ОСМС за себя** — ИП обязан платить ОСМС ежемесячно
3. **Не выставили ЭСФ вовремя** — штраф 15–30 МРП
4. **Неверный ОКЭД** — если деятельность не соответствует ОКЭД, могут снять вычеты
5. **Работали наличными без ККМ** — при превышении лимита 100 МРП/день
6. **Не уведомили об открытии расчётного счёта** — в кабинете налогоплательщика в течение 3 дней
