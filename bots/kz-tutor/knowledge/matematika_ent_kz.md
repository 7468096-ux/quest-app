## 1. АЛГЕБРА

### 1.1 Числа и выражения
- Натуральные, целые, рациональные, иррациональные, вещественные числа
- Степень с целым показателем: a^n · a^m = a^(n+m); (a^n)^m = a^(nm); a^(-n) = 1/a^n
- Корни: √a · √b = √(ab); √a / √b = √(a/b); (√a)² = a (при a ≥ 0)
- Модуль: |a| = a если a ≥ 0; |a| = -a если a < 0

### 1.2 Алгебраические выражения
- Формулы сокращённого умножения:
  - (a+b)² = a² + 2ab + b²
  - (a-b)² = a² - 2ab + b²
  - (a+b)(a-b) = a² - b²
  - (a+b)³ = a³ + 3a²b + 3ab² + b³
  - a³ + b³ = (a+b)(a² - ab + b²)
  - a³ - b³ = (a-b)(a² + ab + b²)

### 1.3 Уравнения

**Линейное уравнение:** ax + b = 0 → x = -b/a (при a ≠ 0)

**Квадратное уравнение:** ax² + bx + c = 0
- Дискриминант: D = b² - 4ac
- D > 0: x₁,₂ = (-b ± √D) / 2a
- D = 0: x = -b / 2a
- D < 0: корней нет
- Теорема Виета: x₁ + x₂ = -b/a; x₁ · x₂ = c/a

**Дробно-рациональное уравнение:**
Пример: (x+1)/(x-2) = 3
1. ODZ: x ≠ 2
2. x + 1 = 3(x-2) → x + 1 = 3x - 6 → -2x = -7 → x = 3.5

**Система двух уравнений:**
- Метод подстановки: выразить одну переменную через другую
- Метод сложения: умножить уравнения и сложить/вычесть
- Пример: {2x + y = 5; x - y = 1} → сложение: 3x = 6 → x = 2, y = 1

### 1.4 Неравенства
- Линейное: ax + b > 0 → x > -b/a (знак меняется при делении на a < 0)
- Квадратное: ax² + bx + c > 0 (или < 0)
  - Найти корни, нарисовать параболу, определить знак
- Метод интервалов для более сложных неравенств

### 1.5 Прогрессии

**Арифметическая прогрессия:**
- aₙ = a₁ + (n-1)d, где d — разность
- Sₙ = n(a₁ + aₙ) / 2 = n(2a₁ + (n-1)d) / 2

**Геометрическая прогрессия:**
- bₙ = b₁ · q^(n-1), где q — знаменатель (q ≠ 0, q ≠ 1)
- Sₙ = b₁(qⁿ - 1) / (q - 1), при q ≠ 1
- Если |q| < 1: S∞ = b₁ / (1 - q)

### 1.6 Степени и логарифмы

**Показательная функция:** y = aˣ (a > 0, a ≠ 1)
- Если a > 1 — возрастающая
- Если 0 < a < 1 — убывающая

**Показательное уравнение:**
- aˣ = aʸ → x = y (если a > 0, a ≠ 1)
- 2^(x+1) = 8 → 2^(x+1) = 2³ → x+1 = 3 → x = 2

**Логарифм:** logₐb = c ⟺ aᶜ = b (a > 0, a ≠ 1, b > 0)
- log₁₀ = lg (десятичный); logₑ = ln (натуральный)
- Свойства:
  - logₐ(mn) = logₐm + logₐn
  - logₐ(m/n) = logₐm - logₐn
  - logₐ(mⁿ) = n·logₐm
  - logₐb = lgb / lga (формула перехода)
- log₂8 = 3 (потому что 2³ = 8)

## 2. ТРИГОНОМЕТРИЯ

### 2.1 Тригонометрические функции
В прямоугольном треугольнике (угол α):
- sin α = противолежащий катет / гипотенуза
- cos α = прилежащий катет / гипотенуза
- tg α = sin α / cos α = противолежащий / прилежащий
- ctg α = cos α / sin α

### 2.2 Основные значения
| α   | 0°  | 30°  | 45°      | 60°      | 90° |
|-----|-----|------|----------|----------|-----|
| sin | 0   | 1/2  | √2/2     | √3/2     | 1   |
| cos | 1   | √3/2 | √2/2     | 1/2      | 0   |
| tg  | 0   | √3/3 | 1        | √3       | —   |

### 2.3 Основные формулы
- Основное тождество: sin²α + cos²α = 1
- 1 + tg²α = 1/cos²α
- 1 + ctg²α = 1/sin²α
- sin(π - α) = sin α; cos(π - α) = -cos α (формулы приведения)
- sin(π + α) = -sin α; cos(π + α) = -cos α
- sin(-α) = -sin α; cos(-α) = cos α

### 2.4 Формулы сложения
- sin(α + β) = sin α cos β + cos α sin β
- cos(α + β) = cos α cos β - sin α sin β
- tg(α + β) = (tg α + tg β) / (1 - tg α · tg β)

### 2.5 Двойной угол
- sin 2α = 2 sin α cos α
- cos 2α = cos²α - sin²α = 1 - 2sin²α = 2cos²α - 1
- tg 2α = 2tg α / (1 - tg²α)

## 3. ФУНКЦИИ

### 3.1 Понятие функции
- f: X → Y, каждому x из X соответствует ровно одно y = f(x)
- Область определения D(f) — допустимые значения x
- Область значений E(f) — множество всех f(x)

### 3.2 Основные функции
| Функция        | Формула   | Область D    | Особенность         |
|---------------|-----------|--------------|---------------------|
| Линейная      | y = kx+b  | ℝ            | k>0 — рост, k<0 — убывает |
| Квадратичная  | y = ax²+bx+c | ℝ         | Парабола, вершина (-b/2a; D/-4a) |
| Корень        | y = √x    | x ≥ 0        | Возрастает          |
| Гипербола     | y = k/x   | x ≠ 0        | k>0 — I и III четв  |
| Показательная | y = aˣ    | ℝ            | a>1 — рост         |
| Логарифмическая | y=logₐx | x > 0        | a>1 — рост         |

### 3.3 Чётность/нечётность
- f(-x) = f(x) → чётная (симметрия оси OY): cos x, x², |x|
- f(-x) = -f(x) → нечётная (симметрия точки O): sin x, x, x³

## 4. ГЕОМЕТРИЯ

### 4.1 Планиметрия

**Треугольник:**
- Сумма углов = 180°
- Площадь: S = (1/2)·a·h = (1/2)·a·b·sin C = √(p(p-a)(p-b)(p-c)) (Герон)
- Теорема Пифагора: a² + b² = c² (c — гипотенуза)
- Теорема косинусов: c² = a² + b² - 2ab·cos C
- Теорема синусов: a/sin A = b/sin B = c/sin C = 2R

**Четырёхугольники:**
- Прямоугольник: S = ab; диагональ d = √(a²+b²)
- Квадрат: S = a²; d = a√2
- Параллелограмм: S = a·h = a·b·sin α
- Трапеция: S = (a+b)/2 · h (a,b — основания)
- Ромб: S = (d₁·d₂)/2

**Окружность:**
- Длина: C = 2πR
- Площадь круга: S = πR²
- Длина дуги: l = αR (α в радианах) = πRα°/180
- Площадь сектора: S = (1/2)R²α = πR²α°/360

### 4.2 Стереометрия

**Прямоугольный параллелепипед (а × b × c):**
- V = abc
- S полн = 2(ab + bc + ac)
- Диагональ: d = √(a²+b²+c²)

**Куб (a):**
- V = a³; S полн = 6a²; d = a√3

**Цилиндр (r, h):**
- V = πr²h; S полн = 2πr(r+h); S бок = 2πrh

**Конус (r, h, l=образующая):**
- V = (1/3)πr²h; l = √(r²+h²)
- S полн = πr(r+l); S бок = πrl

**Пирамида:**
- V = (1/3)·S осн·h

**Шар (R):**
- V = (4/3)πR³; S = 4πR²

## 5. КОМБИНАТОРИКА И ВЕРОЯТНОСТЬ

### 5.1 Комбинаторика
- Перестановки n элементов: Pₙ = n!
- Размещения из n по k: Aₙᵏ = n! / (n-k)!
- Сочетания из n по k: Cₙᵏ = n! / (k!(n-k)!)

**Примеры:**
- В группе 10 человек. Сколькими способами выбрать 3 для команды?
  C₁₀³ = 10! / (3!·7!) = (10·9·8) / (3·2·1) = 120

- n! = 1·2·3·...·n; 0! = 1; 1! = 1; 2! = 2; 3! = 6; 4! = 24; 5! = 120

### 5.2 Теория вероятностей
- Классическая вероятность: P(A) = m/n (m — число благоприятных, n — общее число исходов)
- 0 ≤ P(A) ≤ 1; P(невозможного) = 0; P(достоверного) = 1
- Противоположное событие: P(Ā) = 1 - P(A)
- Независимые события: P(A∩B) = P(A)·P(B)
- Несовместные события: P(A∪B) = P(A) + P(B)

**Типичная задача ЕНТ:**
Из 5 белых и 3 чёрных шаров наугад достают 2. Вероятность, что оба белые?
P = C₅² / C₈² = 10/28 = 5/14 ≈ 0.36
