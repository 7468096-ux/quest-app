### Алгебра и начала анализа

**Тест-2024-М-01**
> Найдите значение выражения: (2³ · 4²) ÷ 2⁵
- A) 4 ✅
- B) 8
- C) 2
- D) 16
- E) 32

*Решение: 2³·4² = 8·16 = 128; 128 ÷ 32 = 4*

---

**Тест-2024-М-02**
> Решите уравнение: 3x² − 12 = 0
- A) x = ±2 ✅
- B) x = ±4
- C) x = 2
- D) x = ±√3
- E) x = ±3

*Решение: x² = 4, x = ±2*

---

**Тест-2023-М-03**
> Функция f(x) = x² − 4x + 3. При каком значении x функция принимает наименьшее значение?
- A) x = 2 ✅
- B) x = 1
- C) x = 3
- D) x = −2
- E) x = 0

*Решение: вершина параболы x = −b/(2a) = 4/2 = 2*

---

**Тест-2025-М-04**
> Чему равен предел: lim(x→∞) (3x² + 5) / (x² − 2)?
- A) 3 ✅
- B) 5/2
- C) 0
- D) ∞
- E) −3

*Решение: делим числитель и знаменатель на x²: (3 + 5/x²)/(1 − 2/x²) → 3/1 = 3*

---

**Тест-2024-М-05**

- D) 9x² − 4x

**Тест-2024-Ф-02**

- E) 0.06 кг

### Частые ошибки на ЕНТ
1. **Математика**: не брать производные от констант (d/dx(C) = 0)
2. **История**: путать даты жузов и ханов — учить по хронологии
3. **Физика**: подставлять единицы СИ (кг, м, с, А)
4. **Чтение**: выбирать ответ из текста, а не общих знаний
5. **Время**: первые 30 вопросов решать быстро (отводить ≤1 мин/вопрос)
