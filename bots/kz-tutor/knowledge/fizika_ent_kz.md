## 1. МЕХАНИКА

### 1.1 Кинематика

**Основные величины:**
- Путь s (м) — суммарное расстояние
- Перемещение s (м) — вектор из начальной в конечную точку
- Скорость v (м/с) — v = s/t
- Ускорение a (м/с²) — a = (v - v₀) / t

**Равномерное движение (a = 0):**
- x = x₀ + v·t
- v = const

**Равноускоренное движение:**
- v = v₀ + a·t
- x = x₀ + v₀·t + a·t²/2
- v² = v₀² + 2·a·s
- Средняя скорость: v_ср = (v₀ + v) / 2

**Свободное падение (a = g = 9.8 ≈ 10 м/с²):**
- Вниз: v = g·t; h = g·t²/2
- Брошено вертикально вверх: v = v₀ - g·t; h_макс = v₀²/2g; t_подъёма = v₀/g

**Равномерное движение по окружности:**
- Период T (с), частота ν = 1/T (Гц)
- Скорость v = 2πR/T = 2πR·ν
- Центростремительное ускорение: a = v²/R = 4π²R/T²

### 1.2 Динамика — Законы Ньютона

**I закон (закон инерции):** Тело сохраняет скорость, если на него не действуют силы (или силы взаимно уравновешены)

**II закон:** F = m·a (равнодействующая = масса × ускорение)
- a = F/m; m = F/a

**III закон:** Сила действия = сила противодействия (противоположны по направлению)
F₁₂ = -F₂₁

**Основные силы:**
- Сила тяжести: F_g = m·g (вниз)
- Нормальная реакция опоры N (перпендикулярна поверхности)
- Сила трения: F_тр = μ·N (μ — коэффициент трения)
- Сила упругости (пружина): F = -k·x (закон Гука; k — жёсткость, x — деформация)

### 1.3 Гравитация
- Закон Всемирного тяготения: F = G·m₁·m₂ / r²
- G = 6.67·10⁻¹¹ Н·м²/кг²
- Ускорение свободного падения: g = G·M_Земли / R_Земли² ≈ 9.8 м/с²

**Первая космическая скорость:** v₁ = √(g·R) ≈ 7.9 км/с
**Вторая космическая скорость:** v₂ = √2 · v₁ ≈ 11.2 км/с

### 1.4 Законы сохранения

**Импульс:** p = m·v (кг·м/с)
- Закон сохранения импульса: p₁ + p₂ = p₁' + p₂' (при отсутствии внешних сил)

**Кинетическая энергия:** E_к = m·v²/2

**Потенциальная энергия:**
- Гравитационная: E_п = m·g·h
- Упругая: E_упр = k·x²/2

**Закон сохранения энергии:** E_к + E_п = const (в замкнутой системе без трения)

**Работа:** A = F·s·cos α (Дж)
**Мощность:** P = A/t = F·v (Вт)
**КПД:** η = A_полезная / A_затраченная × 100%

## 2. МОЛЕКУЛЯРНАЯ ФИЗИКА И ТЕРМОДИНАМИКА

### 2.1 Молекулярно-кинетическая теория
- Вещество состоит из молекул
- Молекулы в непрерывном хаотическом движении
- Молекулы взаимодействуют (притяжение и отталкивание)
- Температура — мера средней кинетической энергии молекул

**Количество вещества:** ν = N/Nₐ = m/M (моль)
- Nₐ = 6.02·10²³ — число Авогадро
- M — молярная масса (кг/моль)

### 2.2 Идеальный газ

**Уравнение состояния:** pV = νRT = nkT
- p — давление (Па), V — объём (м³), T — температура (К)
- R = 8.31 Дж/(моль·К) — универсальная газовая постоянная
- k = 1.38·10⁻²³ Дж/К — постоянная Больцмана
- n = N/V — концентрация молекул

**Перевод температуры:** T(К) = t(°C) + 273

**Изопроцессы (ν = const):**
| Процесс | Условие | Закон |
|---------|---------|-------|
| Изотермический | T = const | pV = const (Бойль-Мариотт) |
| Изобарный | p = const | V/T = const (Гей-Люссак) |
| Изохорный | V = const | p/T = const (Шарль) |

**Пример — Бойль-Мариотт:** p₁V₁ = p₂V₂
Газ сжали с V₁=2л до V₂=0.5л при T=const, p₁=100кПа. Найти p₂:
p₂ = p₁V₁/V₂ = 100·2/0.5 = 400 кПа

### 2.3 Теплообмен

**Количество теплоты:**
- Нагрев/охлаждение: Q = c·m·ΔT (c — удельная теплоёмкость)
- Плавление/кристаллизация: Q = λ·m (λ — удельная теплота плавления)
- Испарение/конденсация: Q = L·m (L — удельная теплота испарения)
- Сгорание: Q = q·m (q — удельная теплота сгорания)

**Уравнение теплового баланса:**
Q_отданная = Q_полученная
c₁m₁(T₁-T) = c₂m₂(T-T₂)

### 2.4 Термодинамика

**I закон термодинамики:** Q = ΔU + A
- Q — полученное теплота; ΔU — изменение внутренней энергии; A — работа газа

**КПД тепловой машины:** η = A/Q₁ = (Q₁-Q₂)/Q₁ = 1 - Q₂/Q₁
- Q₁ — теплота от нагревателя; Q₂ — теплота, отданная холодильнику
- **КПД Карно (максимальный):** η_макс = 1 - T₂/T₁

## 3. ЭЛЕКТРИЧЕСТВО И МАГНЕТИЗМ

### 3.1 Электростатика

**Закон Кулона:** F = k·q₁·q₂/r²
- k = 9·10⁹ Н·м²/Кл²; q — заряды (Кл); r — расстояние (м)

**Электрическое поле:**
- Напряжённость: E = F/q = k·Q/r² (В/м или Н/Кл)
- Потенциал: φ = W/q = k·Q/r (В)
- Напряжение: U = φ₁ - φ₂ = A/q (В)
- Связь E и U (для однородного поля): E = U/d

**Конденсатор:**
- Ёмкость: C = q/U (Ф, фарады)
- Плоский конденсатор: C = ε·ε₀·S/d
- Энергия: W = CU²/2 = qU/2 = q²/2C

### 3.2 Постоянный ток

**Закон Ома для участка цепи:** I = U/R
- U (В), I (А), R (Ом)
- R = ρ·l/S (ρ — удельное сопротивление; l — длина; S — сечение)

**Закон Ома для полной цепи:** I = E / (R + r)
- E — ЭДС источника (В); r — внутреннее сопротивление

**Соединение резисторов:**
- Последовательное: R = R₁ + R₂ + ... ; I = const; U = U₁ + U₂
- Параллельное: 1/R = 1/R₁ + 1/R₂; U = const; I = I₁ + I₂
  - Для двух: R = R₁·R₂/(R₁+R₂)

**Мощность и работа:**
- P = UI = I²R = U²/R (Вт)
- A = P·t = UIt (Дж)
- Тепло (закон Джоуля-Ленца): Q = I²R·t

### 3.3 Магнетизм

**Сила Ампера (ток в поле):**
F = B·I·l·sin α (B — магнитная индукция, Тл; l — длина проводника)

**Сила Лоренца (заряд в поле):**
F = q·v·B·sin α (перпендикулярна v и B)

**Электромагнитная индукция (Закон Фарадея):**
ЭДС = -ΔΦ/Δt (Вольт)
Φ = B·S·cos α — магнитный поток (Вб)

**Самоиндукция:**
ЭДС_саминд = -L·ΔI/Δt
Энергия магнитного поля катушки: W = L·I²/2

## 4. ОПТИКА

### 4.1 Геометрическая оптика

**Скорость света:** c = 3·10⁸ м/с (в вакууме)
**В среде:** v = c/n (n — показатель преломления)

**Закон отражения:** угол падения = угол отражения (оба от нормали)

**Закон Снеллиуса (преломление):** n₁·sin θ₁ = n₂·sin θ₂
- n₁ < n₂: луч преломляется к нормали
- Полное внутреннее отражение: sin θ_кр = n₂/n₁ (при переходе из более плотной среды)

**Линзы:**
- Выпуклая (собирающая): f > 0 (фокусное расстояние положительное)
- Вогнутая (рассеивающая): f < 0
- **Формула линзы:** 1/f = 1/d + 1/v
  - d — расстояние от предмета до линзы
  - v — расстояние от линзы до изображения
  - f — фокусное расстояние (F)
- Оптическая сила: D = 1/f (диоптрии, дптр)
- Увеличение: k = v/d = h'/h

### 4.2 Волновая оптика

**Интерференция:**
- Максимум (усиление): Δ = kλ (k = 0, ±1, ±2, ...)
- Минимум (ослабление): Δ = (2k+1)·λ/2
- λ — длина волны

**Дифракционная решётка:**
d·sin θ = k·λ (d — период решётки, k — порядок)

**Дисперсия:** разложение белого света в спектр (красный — наименьшее преломление; фиолетовый — наибольшее)

## 5. АТОМНАЯ ФИЗИКА И ЯДЕРНЫЕ РЕАКЦИИ

### 5.1 Строение атома
- Модель Бора: электроны на стационарных орбитах
- При переходе E₂ - E₁ = h·ν (h = 6.63·10⁻³⁴ Дж·с — постоянная Планка)
- Фотоэффект: hν = A_вых + E_к(электрона) (уравнение Эйнштейна)

### 5.2 Ядро атома
- Заряд ядра = Z (число протонов) — определяет элемент
- Массовое число A = Z + N (N — нейтроны)
- Нуклоны = протоны + нейтроны

**Радиоактивный распад:**
- α-распад: A → (A-4) + ⁴₂He; Z уменьшается на 2
- β-распад: нейтрон → протон + электрон + нейтрино; Z увеличивается на 1
- γ-излучение: изменения ядра нет, выделяется энергия

**Закон радиоактивного распада:**
N = N₀ · (1/2)^(t/T) = N₀ · e^(-λt)
- T — период полураспада; λ — постоянная распада

**Ядерные реакции:**
- Деление: ²³⁵U + n → ⁹²Kr + ¹⁴¹Ba + 3n + Энергия (цепная реакция)
- Синтез: ²H + ³H → ⁴He + n + 17.6 МэВ (термоядерный синтез)
- Энергия связи ядра: ΔE = Δm · c² (дефект массы)

## 6. КОЛЕБАНИЯ И ВОЛНЫ

### 6.1 Механические колебания

**Гармонические колебания:** x = A·cos(ωt + φ₀)
- A — амплитуда; ω = 2π/T — угловая частота; T — период; ν = 1/T — частота

**Математический маятник:**
T = 2π√(l/g) (l — длина нити, g = 9.8 м/с²)

**Пружинный маятник:**
T = 2π√(m/k) (m — масса, k — жёсткость)

### 6.2 Электромагнитные колебания

**Колебательный контур:**
T = 2π√(LC) (L — индуктивность, Гн; C — ёмкость, Ф)

**Переменный ток:**
- Действующие значения: I_д = I_макс/√2; U_д = U_макс/√2
- Мощность: P = U·I·cos φ

**Электромагнитные волны:**
- Скорость в вакууме: c = 3·10⁸ м/с = λ·ν
- Спектр: радиоволны → микроволны → ИК → видимый → УФ → рентген → γ-лучи

## 7. ТИПОВЫЕ ЗАДАЧИ ЕНТ ПО ФИЗИКЕ

### Тип 1: Кинематика
Автомобиль тормозит с v₀=20 м/с, a=-4 м/с². Найти тормозной путь:
v² = v₀² + 2as → 0 = 400 - 8s → s = 50 м

### Тип 2: Динамика
Тело на горизонтальной поверхности (m=5кг, μ=0.3, g=10). Сила тяги F=20Н. Найти ускорение:
F_тр = μmg = 0.3·5·10 = 15 Н
F_рез = 20 - 15 = 5 Н
a = F/m = 5/5 = 1 м/с²

### Тип 3: Газовые законы
Газ при T₁=300К занимает V₁=2л. При T₂=600К (p=const) найти V₂:
V₁/T₁ = V₂/T₂ → V₂ = 2·600/300 = 4 л

### Тип 4: Электрическая цепь
R₁=4 Ом, R₂=6 Ом подключены последовательно к U=20В. Найти ток и напряжение на R₁:
R = 4+6 = 10 Ом; I = U/R = 20/10 = 2 А; U₁ = I·R₁ = 2·4 = 8 В

### Тип 5: Линза
Предмет на расстоянии d=30 см от линзы с f=20 см. Найти положение изображения:
1/v = 1/f - 1/d = 1/20 - 1/30 = 3/60 - 2/60 = 1/60
v = 60 см (действительное изображение)

### Тип 6: Атомная физика
Период полураспада T=5 лет, начальное количество N₀. Сколько осталось за 15 лет?
t/T = 15/5 = 3; N = N₀·(1/2)³ = N₀/8
