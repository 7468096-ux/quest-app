## 1. NUMBER THEORY & ARITHMETIC

### Number Sets
- Natural numbers: {1, 2, 3, ...}
- Whole numbers: {0, 1, 2, ...}
- Integers: {..., -2, -1, 0, 1, 2, ...}
- Rational numbers: can be written as p/q where q≠0 (includes fractions, decimals that terminate or repeat)
- Irrational numbers: cannot be expressed as fraction (π, √2, √3, e)
- Real numbers: all rational + irrational numbers

### Divisibility Rules
- By 2: last digit is even (0,2,4,6,8)
- By 3: sum of digits divisible by 3 → 123: 1+2+3=6 ✓
- By 4: last two digits divisible by 4 → 316: 16÷4=4 ✓
- By 5: last digit is 0 or 5
- By 6: divisible by both 2 and 3
- By 9: sum of digits divisible by 9 → 729: 7+2+9=18 ✓
- By 10: last digit is 0

### GCF and LCM
- **GCF (Greatest Common Factor)**: largest number dividing both
  - Method: prime factorization → multiply COMMON factors with SMALLER exponents
  - GCF(12, 18): 12=2²×3, 18=2×3² → GCF=2×3=**6**
- **LCM (Least Common Multiple)**: smallest number divisible by both
  - Method: prime factorization → multiply ALL factors with LARGER exponents
  - LCM(12, 18): 12=2²×3, 18=2×3² → LCM=2²×3²=**36**

### Fractions, Decimals, Percents
- Fraction → Decimal: divide numerator by denominator (3/4 = 0.75)
- Decimal → Percent: multiply by 100 (0.75 = 75%)
- Percent → Fraction: divide by 100 (75% = 75/100 = 3/4)
- **Percentage problems**: Part = Rate × Base
  - What is 30% of 150? Part = 0.30 × 150 = 45
  - 45 is what percent of 150? Rate = 45/150 = 0.30 = 30%
  - 45 is 30% of what? Base = 45/0.30 = 150

## 2. ALGEBRA

### Linear Equations
- Standard form: ax + b = c → solve for x: x = (c-b)/a
- Example: 3x + 7 = 22 → 3x = 15 → x = 5

### Systems of Linear Equations
**Substitution method:**
- 2x + y = 10 and x - y = 2
- From 2nd: x = y + 2
- Substitute: 2(y+2) + y = 10 → 2y+4+y = 10 → 3y = 6 → y = 2, x = 4

**Elimination method:**
- 2x + 3y = 12 and 4x - 3y = 6
- Add equations: 6x = 18 → x = 3
- Substitute: 2(3) + 3y = 12 → 3y = 6 → y = 2

### Quadratic Equations (ax² + bx + c = 0)
**Factoring method:** x² - 5x + 6 = 0 → (x-2)(x-3) = 0 → x = 2 or x = 3

**Quadratic Formula:**
x = (-b ± √(b² - 4ac)) / 2a

**Discriminant (D = b² - 4ac):**
- D > 0: two distinct real roots
- D = 0: one real root (repeated)
- D < 0: no real roots (complex)

Example: x² - 5x + 6 = 0 → a=1, b=-5, c=6
x = (5 ± √(25-24)) / 2 = (5 ± 1) / 2 → x = 3 or x = 2 ✓

### Polynomials
- Degree = highest exponent
- Sum of roots = -b/a; Product of roots = c/a (for ax²+bx+c=0)
- FOIL: (a+b)(c+d) = ac + ad + bc + bd
- Special products:
  - (a+b)² = a² + 2ab + b²
  - (a-b)² = a² - 2ab + b²
  - (a+b)(a-b) = a² - b²
  - (a+b)³ = a³ + 3a²b + 3ab² + b³

### Functions and Relations
- Function: each input (x) has exactly ONE output (y)
- Vertical Line Test: if any vertical line crosses graph more than once → NOT a function
- Domain: set of all valid x values
- Range: set of all possible y values
- f(x) notation: f(x) = 2x + 3 means y = 2x + 3

### Inequalities
- Flip the inequality sign when multiplying/dividing by a NEGATIVE number
- Example: -2x > 6 → x < -3 (sign flipped!)
- Graph on number line:
  - ≤ or ≥: closed circle (●)
  - < or >: open circle (○)

## 3. GEOMETRY

### Angle Relationships
- Complementary angles: sum = 90°
- Supplementary angles: sum = 180°
- Vertical angles: equal (opposite angles when two lines cross)
- Linear pair: supplementary (adjacent angles on a straight line)
- Parallel lines cut by transversal:
  - Alternate interior angles: equal
  - Corresponding angles: equal
  - Co-interior (same-side interior) angles: supplementary (sum=180°)

### Triangle Properties
- Sum of interior angles = 180°
- Exterior angle = sum of two non-adjacent interior angles
- Triangle Inequality: sum of any two sides > third side
- Types by sides: equilateral (all equal), isosceles (2 equal), scalene (all different)
- Types by angles: acute (all <90°), right (one =90°), obtuse (one >90°)

**Special Right Triangles:**
- 30-60-90: sides in ratio 1 : √3 : 2
  - Short leg = x, long leg = x√3, hypotenuse = 2x
- 45-45-90: sides in ratio 1 : 1 : √2
  - Both legs = x, hypotenuse = x√2

**Pythagorean Theorem:** a² + b² = c² (c = hypotenuse)
Common Pythagorean triples: 3-4-5, 5-12-13, 8-15-17, 7-24-25

### Formulas — Plane Figures
| Shape | Perimeter | Area |
|-------|-----------|------|
| Square (s) | 4s | s² |
| Rectangle (l×w) | 2(l+w) | lw |
| Triangle (base b, height h) | a+b+c | ½bh |
| Circle (radius r) | 2πr | πr² |
| Parallelogram | 2(a+b) | bh |
| Trapezoid | a+b+c+d | ½(b₁+b₂)h |

### Formulas — Solid Figures
| Shape | Surface Area | Volume |
|-------|-------------|--------|
| Cube (s) | 6s² | s³ |
| Rectangular prism (l,w,h) | 2(lw+lh+wh) | lwh |
| Sphere (r) | 4πr² | (4/3)πr³ |
| Cylinder (r,h) | 2πr²+2πrh | πr²h |
| Cone (r,h,l) | πr²+πrl | (1/3)πr²h |
| Pyramid (base B, height h) | B + lateral faces | (1/3)Bh |

### Coordinate Geometry
- Distance formula: d = √((x₂-x₁)² + (y₂-y₁)²)
- Midpoint formula: M = ((x₁+x₂)/2, (y₁+y₂)/2)
- Slope: m = (y₂-y₁)/(x₂-x₁)
- Slope-intercept form: y = mx + b (m=slope, b=y-intercept)
- Parallel lines: equal slopes (m₁ = m₂)
- Perpendicular lines: negative reciprocal slopes (m₁ × m₂ = -1)

## 4. TRIGONOMETRY

### Basic Ratios (SOH-CAH-TOA)
For a right triangle with angle θ, hypotenuse H, opposite O, adjacent A:
- **sin θ = O/H** (Sine = Opposite over Hypotenuse)
- **cos θ = A/H** (Cosine = Adjacent over Hypotenuse)
- **tan θ = O/A** (Tangent = Opposite over Adjacent)
- csc θ = H/O (reciprocal of sin)
- sec θ = H/A (reciprocal of cos)
- cot θ = A/O (reciprocal of tan)

### Special Angle Values
| θ | sin θ | cos θ | tan θ |
|---|-------|-------|-------|
| 0° | 0 | 1 | 0 |
| 30° | 1/2 | √3/2 | 1/√3 = √3/3 |
| 45° | √2/2 | √2/2 | 1 |
| 60° | √3/2 | 1/2 | √3 |
| 90° | 1 | 0 | undefined |

### Pythagorean Identity
sin²θ + cos²θ = 1 (most important identity)
Also: 1 + tan²θ = sec²θ and 1 + cot²θ = csc²θ

## 5. STATISTICS & PROBABILITY

### Measures of Central Tendency
- **Mean**: sum of values ÷ number of values
  - Example: {4,7,9,3,7} → mean = (4+7+9+3+7)/5 = 30/5 = **6**
- **Median**: middle value when arranged in order
  - Odd count: middle value → {3,4,7,7,9} → **7**
  - Even count: average of two middle values → {2,4,6,8} → (4+6)/2 = **5**
- **Mode**: most frequent value → {3,4,7,7,9} → **7**
- **Range**: max - min → 9-3 = **6**

### Counting Principles
- **Fundamental Counting Principle**: if event A has m ways and event B has n ways → m × n total ways
  - Example: 3 shirts × 4 pants = 12 outfits
- **Permutation** (order matters): P(n,r) = n! / (n-r)!
  - Arranging 3 from 5: P(5,3) = 5×4×3 = 60
- **Combination** (order doesn't matter): C(n,r) = n! / (r!(n-r)!)
  - Choosing 3 from 5: C(5,3) = 10

### Basic Probability
- P(event) = favorable outcomes / total possible outcomes
- 0 ≤ P(E) ≤ 1
- P(not E) = 1 - P(E)
- P(A or B) = P(A) + P(B) - P(A and B) [Addition Rule]
- P(A and B) = P(A) × P(B) [if independent events]
- Example: Rolling a die, P(even) = 3/6 = 1/2

## 6. WORD PROBLEMS STRATEGIES

### Step-by-Step Approach
1. **READ** the problem twice
2. **IDENTIFY** what you're looking for (assign a variable: let x = ?)
3. **TRANSLATE** words to math expressions
4. **SOLVE** the equation
5. **CHECK** if the answer makes sense

### Common Word Problem Types

**Age Problems:**
- "Juan is 5 years older than Maria. 3 years ago, the sum of their ages was 25. Find their current ages."
- Let Maria's age = x, Juan's age = x+5
- 3 years ago: (x-3) + (x+5-3) = 25 → 2x-1=25 → x=13, Juan=18

**Distance-Speed-Time:**
- d = s × t (distance = speed × time)
- Two cars travel toward each other: d₁ + d₂ = total distance
- Same direction: d₁ - d₂ = difference

**Mixture Problems:**
- Amount₁ × Concentration₁ + Amount₂ × Concentration₂ = Total × Final Concentration

**Work Problems:**
- If A does work in 'a' hours and B in 'b' hours, together: 1/a + 1/b = 1/t
- Time together: t = ab/(a+b)

## PRACTICE PROBLEMS

**Q1 (Algebra):** Find the roots of x² - 7x + 12 = 0
Solution: Factor → (x-3)(x-4) = 0 → **x = 3 or x = 4**

**Q2 (Geometry):** A circle has circumference 20π cm. What is its area?
Solution: C = 2πr → 2πr = 20π → r = 10 cm → A = πr² = **100π cm²**

**Q3 (Statistics):** The mean of 5 numbers is 8. If four of the numbers are 5, 9, 11, and 7, what is the fifth number?
Solution: Sum = 8×5 = 40; Sum of four = 5+9+11+7 = 32; Fifth = 40-32 = **8**

**Q4 (Probability):** A bag has 3 red, 4 blue, 5 green balls. P(not green) = ?
Solution: Total = 12; Not green = 7; P = 7/12

**Q5 (Word Problem):** Two trains leave the same station in opposite directions. Train A at 80 km/h, Train B at 100 km/h. After how many hours are they 540 km apart?
Solution: Combined speed = 180 km/h; t = 540/180 = **3 hours**
